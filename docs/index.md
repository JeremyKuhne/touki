# Touki Documentation

Welcome to the Touki docs.

Touki is a .NET and .NET Framework utility library focused on:

- Avoiding unnecessary allocations
- Avoiding code that prevents AOT compilation on .NET
- Leveraging modern C# features for ergonomics and performance
